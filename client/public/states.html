Available API Endpoints

GET /api/states - List all states
GET /api/states/:code - Get specific state data
GET /api/states/:code/sources - Get state source URLs
PUT /api/states/:code - Update state research data (requires auth)
POST /api/states/:code/research - Run assisted research (requires auth)
GET /api/export?type=states - Export all states research
GET /api/export?state=:code - Export single state research
